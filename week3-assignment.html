「글래스 코드」 소문 확산 — 매체 실행

3주차 과제 = 매스 1가지 + 온 1가지. (강사: 콘텐츠 지시서는 개인 과제로) · 조장 김서아 · 배영순 · 하지우 · 박지호

12주차 발표 피드백 반영 (수정)

3주차 발표 때 받은 강사님 피드백을 먼저 반영해 다듬었습니다.

2주차 (수정 전)소문 문장 “잔 하나로 취향을 읽고 대화의 문을 여는 바” — 길고 설명적
3주차 (수정 후) ⭐소문 문장 “잔으로 인연을 만드는 곳.” (강사 추천 — 더 짧고 직관적)
  • 오브제 디벨롭 — 취향 카드(입장권)에 더해, ‘매칭 카드(찢어 나눠 갖기)’ 추가. 둘만 맞춰지는 상징물 → 소문거리. (강사 추천)
  • 운영 디테일 — 스탠딩 바(앉은 자리에서 옮기기 부담 ↓) + 드레스 코드(반바지·슬리퍼·나시 X) + 잔 가격으로 격(퀄리피케이션) 유지.
  • 논란거리 = 욕망 자극 — “싱글만?·유부남녀는?” 같은 갑론을박이 오히려 소문이 된다(강사: 시장에 전달되면 무조건 잘 됨).

2회의에서 정한 것 (요약)

매스 = 공간 매체 · 테이블 안내문  /  온 = 취미 동호회 ‘호스트’ 콜라보
  • 매스는 5가지(검색·공간·상품·사람·콘텐츠) 중 공간 매체를 선택 — 우리는 ‘취향 카드’를 곳곳에 세우니 ‘테이블 안내문’이 가장 자연스러운 매체.
  • 온은 핵심 타겟(30대 후반~40대 후반 싱글)이 속한 취미 동호회/소모임의 호스트(주최자)를 통해 확산.

3과제 ① 매스 — 공간 매체 : 테이블 안내문

스탠딩 바 곳곳(테이블·주문대)에 세워 둔다. 안내문이 ‘말 거는 법·거절하는 법·매칭 카드 쓰는 법’을 비대면으로 안내 = 손님이 부담 없이 글래스 코드의 룰을 이해하게 만드는 매체.

「글래스 코드」 — 잔으로 인연을 만드는 곳
  • 같은 취향(잔)이 통하는 분께, 편하게 말을 걸어보세요.
  • 대화가 부담스러우면, 예의 있게 거절하셔도 괜찮습니다. (혼자 있고 싶은 날엔 카드를 뒤집어 두세요.)
  • 마음이 통했다면 — 연락처 대신 이 매칭 카드를 찢어, “다음에 여기서 다시 만나요”(날짜·시간)를 나눠 가지세요.
  • 직접 말 걸기 어려우면, 직원에게 말씀해주세요.
＋ 사람 매체(직원)와 연결 — 직원은 처음 와서 어색한 손님께 “오늘 ○○ 취향 분들 계세요” 식으로 프레이밍하고, 매칭 카드(찢어 나눠 갖기) 사용법을 한 번 더 안내. (강사: 직원 = 시장과 실시간 소통하는 유일한 매체 = 통역가)

4과제 ② 온 — 취미 동호회 ‘호스트’ 콜라보

핵심 타겟
사회적으로 자리 잡은 30대 후반~40대 후반 싱글.
속한 집단
취미 동호회·소모임(골프·와인·러닝·스쿠버 등) — 우리 ‘취향 칵테일’과 결이 같고, 솔로가 많으며 공감대 기반이라 적합.
의사결정권자
동호회/소모임의 ‘호스트(모임 주최자)’. (‘소모임’ 앱에서 러닝·와인 모임을 여는 사람처럼 — 여론을 주도)
호스트의 욕망
좋은 신규 회원 영입 + 모임 분위기·뒤풀이(번개) 장소.
우리의 욕구
품격 있는 진짜 솔로 남녀 유입 + 재방문.

→ 두 욕구를 묶은 콜라보 제안

  • “동호회 단체로 오면 혜택 + 같은 취향끼리 자연스럽게 어울리는 곳(모임 뒤풀이·번개 장소)으로 쓰세요.”
  • 호스트는 회원에게 입소문을 내고, 우리 바에서 새 회원도 영입(공감대 맞는 사람) → 호스트·우리 둘 다 이득.
  • 분기별 1회 이상, 여러 동호회(골프·와인·러닝…) 호스트에게 반복 제안. (강사: 온은 한 번이 아니라 계속)
확장 — 협업은 ‘단체 동호회’ 단위로 디벨롭 가능(소모임·넷플연가 같은 호스트 플랫폼과). 단체 방문 혜택을 미끼로 호스트를 매체화.

＋개인 과제 (각자 매장)

자기 업에 매스 1가지 + 온 1가지 + 콘텐츠 지시서(5가지)까지. (전파할 소문 문장 / 느낄 감정 / 보여줄 증거 / 금지 표현 ‘맛있다X→다르다O’ / 행동 제한)
