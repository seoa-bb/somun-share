제갈현열 작가의 소문설계 1기 · 3주차 · 6개 조 발표 피드백

2주차 과제 발표 — 6개 조 피드백 정리

발표 = 가치 선언문 · 소문 문장 · 오브제/이미지. (이번엔 1→6조 순서)

★강사님 전체 관통 피드백

좋은 소문거리의 판단 기준 = “선이 그어지냐.”
  • ‘더 낫다’(원 오브 뎀) X → ‘다르다’(다른 카테고리) O. 두부 ‘맛있는’ 집(베터 싸움) ❌ → 부모님 ‘칭찬받는’ 집(다른 카테고리) ⭕.
  • 선이 그어지면 소비자 머릿속에서 기존 것들이 한 덩어리가 되고 우리만 따로 선다 → 소문이 난다.
  • 이번 6개 조 중 ‘더 낫다’로 온 조가 한 명도 없음 = 고무적. (보통은 다 “우리 건 더 낫다”고 함)
  • ⭐ 오브제는 ‘선물’이 목적이 아니다. 받은 사람이 그걸 구실로 ‘남에게 소문낼 수 있어야’ 한다. (가족사진 액자처럼 집에 걸려 끝나면 X)
  • 소문 문장은 ‘카피라이팅’이 아니다. 그대로 전달될 가능성은 0 — 시장이 각자 ‘해석’해 전달한다. 잘 표현했냐보다 ‘옮길 만한가’가 중요.
  • 좋은 소문거리는 머리로 ‘납득’보다 대상의 ‘욕망’을 파고든다.

1조콩·수제 두부 프리미엄 다이닝

컨셉 3대가 함께하는 품격 있는 화목한 시간 (40대 주부가 부모+아이 모시는 자리)

소문 문장 “부모님 모시고 가면 칭찬받는 두부집”

오브제 냅킨 골드링 + 네임 명패 / 가족사진 액자

강사 피드백
  • 소문 문장 매력적 — ‘부모님 칭찬받는’이 욕망을 파고듦(공감). 시부모면 더 극적(“시월드 분쇄 맛집”).
  • 본질 = ‘맛있다/품격’이 아니라 “부모님 모시고 갔을 때 칭찬받는다”는 시장이 원하는 얘기.
  • ⚠️ 가족사진 액자 = 격에 안 맞고 패밀리레스토랑 폴라로이드와 비교됨 → 소문 안 남.
  • → 가족사진을 ‘자랑할 물건’으로: 날짜 라벨링한 하우스 와인 선물. 집에서 친구에게 자랑하며 마심 → 소문 + 와인 떨어지면 재방문.
  • 명패는 OK(특이해서 찍어 소문). 진짜 나무로 예쁘게.

2조글래스 코드 — 우리 조 ⭐

시장 사회적으로 자리 잡은 30대 후반~40대 후반 싱글

핵심 “품격 있게 다가가고, 예의 있게 거절하는 곳” (만남은 잠재욕구에 감춤)

소문 문장 “잔 하나로 취향을 읽고 대화의 문을 여는 바”

오브제 취향 카드(골프·와인·트래블) = 칵테일 주문이 곧 취향 인증·입장권

강사 피드백
  • 1주차 피드백(뾰족하게·시장 좁혀라)을 가장 충실히 반영. 사용법 설명 좋음.
  • 도구임을 어떻게 표현? → 네이버 플레이스 소개글에 가치를 소비자 언어로 (예: “품격 있게 매칭되는 취향 바”).
  • 소문 문장이 길다 → 더 짧고 직관적으로 “잔으로 인연을 만드는 곳” 추천. (“헌팅은 없지만”은 굳이 격 낮출 필요?)
  • 논란거리(유부남녀·싱글만?)가 욕망을 건드림 → 시장에 전달되면 갑론을박 → 무조건 잘 됨(대박).
  • 실제 운영: 스탠딩 테이블 추천(앉아서 다른 자리 가기 어려움). 룸 X. 잔 가격 ↑ + 드레스 코드 필수(반바지·슬리퍼·나시 입장 불가).
  • ⚠️ 오브제 고민 필요(고급 바 사진은 못 올림) → 추천: ‘찢는 종이’ — 매칭되면 종이를 찢어 둘이 나눠 가짐. ‘이거 뭐야?’ 대화를 유발하는 둘만의 상징물 → 소문거리.

→ 우리 조가 반영할 점

  • 소문 문장 후보에 “잔으로 인연을 만드는 곳” 추가(짧고 강함).
  • 오브제: ‘취향 카드’에 더해 ‘둘만 나눠 갖는 매칭 오브제(찢는 종이/태그)’ 디벨롭.
  • 운영 디테일: 스탠딩 중심 + 드레스 코드 + 잔 가격 퀄리피케이션.

3조대구 약기우동 (주민증 오브제)

컨셉 “우리가 알던 약기우동이 아니네” — 일식 아닌 대구식 빨간 중식

오브제 약기우동 ‘주민등록증’(베네통식 비틀기 — 음식을 사람처럼, 대구 시민권·호적)

강사 피드백
  • 오브제(주민증)의 의식의 흐름·확장이 신선. 베네통식 관점 전환 OK.
  • ⚠️ “일식 아닌 빨간 중식”이 시장이 ‘올 이유(설득력)’가 되나? 한 단계 생략됨 — 소구는 되나 설득은 약함.
  • 대구 사람에게 “빨간 중식이라 주장해라”론 안 옴. 대구 밖에서 “네가 먹던 대구 찐맛 여기 있다”가 차별점. (뭉티기 = 뒤집는 비주얼 쇼크로 서울 확산, 본점 경상도)
  • 소문 문장·컨셉 재고 + 신분증에 ‘무슨 장치를 넣어야 찍어 소문낼까’ 고민하면 디벨롭됨.

4조썸타임 — MBTI 타르트

컨셉 썸타는 2030이 상대 MBTI에 맞는 타르트 선물. “네 글자의 설렘”

소문 문장 “썸타임, 그녀를 찾아주세요”

오브제 MBTI 모양 타르트(맛 다르게) + 예쁜 포장

강사 피드백
  • 방향성·아이템 좋음. MBTI는 ‘문화’라 오래 감(혈액형 30년·MBTI 20년) → 선점 좋다.
  • ⚠️ 올리는 건 ‘주는 남자’가 아니라 ‘받는 여자’(자랑) → 타겟·소문 문장을 받는 쪽(“썸 당하고 있다”)으로. (건축학개론 “우리 모두는 누군가의 첫사랑” 결)
  • ‘썸 선물’로는 MBTI 핍진성 약함 → 타겟 확대: “가까워지고 싶은 사이” / 비즈니스(I know you, 상대 MBTI 기억=친해지는 용). 그러면 타르트 자체가 오브제 + UGC.
  • (참고: MBTI 키링 = 최애 아이돌+자기 MBTI·팀컬러 LED = 유사연애·비밀연애 코드)

5조부모–아이 커플룩 의류

잠재욕구 (젊어 보이고 싶다 →) 아이와 친해지고 화목한 가정

소문 문장 “애가 엄마랑 똑같이 입고 싶어 하는 거, 그거 지금뿐이야” (타임어택)

오브제 (미완) 이미지 = 커플룩 한 장의 추억

강사 피드백
  • 소문 문장 잘 뽑음 — ‘품 안의 자식’ 욕망 + 타임어택(마감 전략)으로 구매 자극.
  • ⚠️ “왜 엄마?” 엄마-딸 커플룩은 흔함 → 부모로 확대(아빠-아들/딸은 드뭄). 엄마는 ‘예쁜 게’, 아빠는 ‘웃긴 게’ 나옴 → 아빠 패셔니스타 컨셉이 구매욕 자극.
  • 오브제 추천: 이벤트용 풍선(하루 쓰는, 사람 많은 곳에서 부모-자식 확인용, 하트+가게 상호) → 뒷모습이 귀여워 소문.

6조온마 — 반찬·도시락 구독 (동네 맛집 큐레이션)

타겟 엄마(구매권). “충분히 좋은 엄마인데, 매일 요리하지 않아도 되는 ‘좋은 엄마의 밥상’”

장치 맛집 지도 어플(포켓몬고식) + 영양사 카드

강사 피드백
  • ⚠️ 동네 맛집 큐레이션 = 결국 ‘외부 음식만 먹는다’는 인상 → 엄마의 죄책감 해결 안 됨.
  • 단순 배달이면 영양사가 왜 필요? 기능 가치(설득 포인트)가 부족.
  • → 장치 추가: 저염식·건강 기준으로 동네 음식점 선별 + 3개월마다 리밸런싱, 커스터마이징(염도↓). “우리가 있어서 네가 엄마 역할을 제대로 할 수 있다”가 돼야.
  • 오브제: 아이가 자랑할 굿즈(구독 시) → 아이가 친구에게 자랑 → 부모에게 조름(마트 시식코너 원리).
