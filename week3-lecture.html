제갈현열 작가의 소문설계 1기 · 3주차 강의 정리

소문 확산 실행 — “매체”

1·2주차에 만든 소문거리를 시장에 ‘전달’하는 단계. (강사: 4주 중 가장 쉬운 주차)

0들어가며 — 기량(패시브 스킬)

  • 1주차 = 소문거리 개발(누구에게·어떤 욕구·어떤 가치) / 2주차 = 가치 선언문 → 소문 문장 → 오브제·이미지 / 3주차 = 이걸 시장에 전달.
  • 2주차가 제일 어려운 이유: 이 사이클을 ‘자기 꼴’로 만들어야 3·4주차 도구를 활용할 수 있다. (외주로 만들려면 제대로 된 건 최소 2억. 낮은 금액 = 템플릿 컨설팅)
  • 기량 = 운전·자전거처럼 할수록 쌓이고, 한번 쌓이면 되돌아갈 수 없는 패시브 스킬. 기량이 생기면 모든 마케팅 툴이 ‘도구’가 되고, 없으면 툴에 ‘의존’하게 된다.

1매체란 — 개념의 확장

매체 = 내 이야기와 시장(소비자)이 만나는 모든 접점·순간.
  • 매체를 ‘TV·잡지·인스타’ 같은 도구로 보지 말 것. 개념을 확장하는 것 — 내 소문거리와 타겟이 만나는 지점을 떠올리면 전부 매체가 된다.
  • 매체에 좋고 나쁨은 없다. 내 이야기에 ‘맞냐 안 맞냐’만 있다.
IWC 버스 손잡이“시계는 차면 예쁘다 → 사람이 손목에 뭘 차는 순간이 언제지?” → 버스 손잡이를 시계 광고로. 아무도 매체라 안 하던 손잡이가 매체가 됨. (반대로 ‘대리운전’ 버스손잡이 광고 = 이야기와 안 맞는 잘못된 매체)

2두 갈래 — 매스(Mass) vs 온(On)

매스 = 낚시
내 가게를 둘러싼 모든 사람에게 노출시키고, 소문거리가 퍼지길 기다린다.
온 = 사냥
한 사람(영향력자)을 찍어 그를 통해 내 소문이 시장에 퍼지게 만든다.

3매스 — 5가지 매체

자영업자라면 누구나 이미 가지고 있는 접점들. 전부 하라는 게 아니라 ‘내 업에 맞는 것’을 골라 소문거리를 입힌다. (강사 예시 = 가상의 ‘5시간 4번 물기 빼는 540 반숙성회’ 횟집)

① 검색 매체 — 이용 ‘전’에 마주하는 곳

  • 네이버 플레이스·카카오맵·구글맵·블로그·인스타 검색 등. 가게의 첫인상.
  • 소개글에 차별점을 한 마디로(예: “5시간 4번 물기 빼는 540 기법, 반숙성회 전문점”). 평이한 설명 = 평범 = 소문 안 남.
  • 사진은 메뉴·매장만 X → 소문거리를 보여주는 이미지도. ⚠️ AI 사진 절대 금지(급 떨어짐). 줄 선 사진은 모자이크/45도 뒷각도로.

② 공간 매체 — 매장에서 ‘시선이 머무는’ 모든 곳

  • 간판·입구문구·수조·숙성냉장고·바테이블·벽면·메뉴판·테이블 안내문·계산대·화장실까지.
  • 간판 서브카피: 꾸미는 단어 X → 소문거리로. 멋있을 필요 없음(경험 후 맴돌면 됨). “70세 할머니 증명사진” 간판이 더 소문남.
  • 입구문구: 읽게 X, 보게 O(짧게). 스치듯 지나가니까.
  • 메뉴판: 정보 X, 의도 O. (절창 = ‘씹을수록 고소한 특수부위’ / 숙성육관 = 웻에이징 vs 드라이에이징 가격차로 차별 전달) → 540 vs 320 가격차로 나누면 ‘돈 더 줄 만큼 특별’ 전달.
  • 벽면문구: 크니까 사진 찍어 올리기 좋은 대형 광고판 → 가치 선언문(전달용)을 적는다. “고기는 살, 안 살은 내가” 같은 위트 X.
  • 테이블 안내문: 비대면으로 ‘딜(제안)’할 수 있는 공간. 리뷰 이벤트는 방향 제한(“540 반숙성회의 찰기·감칠맛 시식을 남겨주세요”) → 맥락이 모인 리뷰 = 강력한 검색 매체.

③ 상품 매체 — 구매한 상품 ‘자체’가 매체

  • 시장이 산 상품도 매개체. (뭉티기 = 제품 자체가 소문거리)
  • 퍼포먼스로 소문거리 직관화: 회 뒤집어도 안 떨어짐 / 소금이 안 녹음(물기 다 빠진 증거) / 의미를 담은 반찬 구성.

④ 사람 매체 — 직원 = 유일하게 ‘실시간 소통’하는 매체

  • 직원은 소문거리를 고객 경험으로 옮기는 ‘통역가’.
  • 프레이밍 이론: 같은 ‘에쿠스’도 선행 자극에 따라 ‘기름 많이 먹는 차/안전한 차’로 갈림. → 경험 ‘전에’ 틀을 지어주면 해석 방향이 정해진다.
  • (절창집: 메뉴판 + 직원이 한 번 더 “씹을수록 고소한 부위” 말하니 ‘질기다’ 불만 10/10 → 0)
  • 친절·태도만 교육 X → 소문거리를 프레이밍하게 교육. (실수는 OK, 교육받은 걸 ‘귀찮아서 안 하는 건’ X)

⑤ 콘텐츠 매체 — 유일하게 ‘돈 주는’ 매체

  • 릴스·블로그·유튜브 쇼츠·틱톡 외주. “알아서 잘해주세요” = 가장 무책임. (업체의 “오랜 노하우로 알아서” 도 무책임)
콘텐츠 지시서 — 최소 5가지를 내가 지시해야 돈값을 한다
  • ① 이번에 전파할 소문 문장 (의미가 들어가게)
  • ② 시장이 느껴야 할 감정
  • ③ 반드시 보여줄 증거(오브제·이미지) 중 최소 2가지
  • ④ 절대 쓰면 안 되는 표현(싱싱·가성비·푸짐·친절 / “맛있다” X → “맛이 다르다” O)
  • ⑤ 보는 사람의 행동 제한(예: “회부심 있으면 지인 데려가 자랑하세요”)

4팩트 vs 픽션

  • 픽션(안 하는데 한다고) = 잘됐을 때 공격받는다 → ❌ (나락 연예인처럼 ‘잘될 때’ 한 방에).
  • 논쟁거리(말은 되는 주장)는 OK. 단 ‘내피셜’ X — 거짓이 아니고, 내가 실제로 그 행위를 하고 있어야 한다.
런던 베이글 · 평양냉면“런던엔 저런 베이글 없다 / 평양냉면 원래 안 밍밍하다” 악플이 달려도 장사 잘 됨. 유명한 소문엔 팩트체크가 따라붙지만 대부분 소문을 못 이긴다. (단, 앞뒤 다른 모습만 안 보이면 됨)

5온(On) 마케팅 — 전염병

핵심 타겟 ‘한 사람’을 매체로 만들어, 유사 타겟에게 기하급수로 퍼뜨린다.

프로세스

  • ① 타겟 페르소나 → 속한 집단 → 의사결정권자(여론 주도자) 선정. (예: 회부심 3545 남자 → 낚시 동호회/회 유튜브/결정사 → 동호회장·수산물 유튜버·운영진)
  • ② 그 의사결정권자의 ‘욕망’ + 나의 ‘욕구’를 결합해 제안(콜라보)을 만든다. (예: 수산물 유튜버는 ‘양질 콘텐츠’에 목마름 → ‘회물레 챌린지’ 제안 → 그가 인정하면 그 영상 보는 타겟이 몰려옴)
  • ③ 분기별 최소 1회는 시도. (소문은 영속이 있어야 하니 같은 자극을 계속)
컨택 3가지 — 네트워크 / 콜드메일(100개 보내면 2개 답장, 그 2개가 계기) / 자기 채널(4주차). “그 사람 몰라요”는 핑계 — 없으면 만든다. (강사: 책 쓸 때 시장·장관·금메달리스트 다 콜드/소개로 만남)

63주차 과제 안내

  • 조별: ① 매스 1가지(간판 등 아무거나) 적용 + ② 온 프로세스에 따른 제안 1개 구성.
  • 개인: 매스 1가지 + 온 1가지 + 콘텐츠 지시서(5가지)까지.
관통하는 한 줄 — 소문거리는 ‘더 나은 이야기’가 아니라 ‘선이 그어지는 다른 이야기’로 만든다. 매체는 그 한 가지 핵심 소문 문장을, 접점마다 형태만 바꿔 전달하는 일.
